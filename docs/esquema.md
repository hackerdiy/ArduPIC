# Conexión del PIC al Arduino

## Para DIP 28 (FAMILIA 18F2XXX)

| ARDUINO | PIC |
| ------- | --- |
| D5      | 28 - PGD  |
| D6      | 27 - PGC  |
| D7      | 1 - MCLR  |
| D8      | 26 - PGM  |
| 5V      | 20 - VDD  |
| GND     | 12 - VSS  |
| GND     | 8 - VSS   |

![Conexión DIP 28](/ima/dip28.png)
